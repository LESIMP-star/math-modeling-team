# 数模知识库

此目录存放拆解后的数学建模竞赛优秀论文，供 `/math-model` skill 检索使用。

## 如何添加论文

### 方法一：使用 /math-papers 自动拆解
```
/math-papers /path/to/paper.pdf
```
会自动将 PDF 拆解为结构化 .md 文件存入此目录。

### 方法二：手动添加
按照 `sample.md` 的格式手动编写。

## 文件命名规范

```
{年份}_{比赛}_{关键词}.md
```

示例：
- `2023_国赛_城市交通流量预测.md`
- `2022_国赛_中药材定价模型.md`
- `2024_电工杯_电力负荷预测.md`

## 标签规范

### 问题类型标签
- `#评价模型` — 综合评价、排序类
- `#预测模型` — 预测、趋势分析类
- `#优化模型` — 优化、调度、规划类
- `#分类模型` — 分类、聚类、判别类
- `#统计分析` — 统计检验、回归分析类
- `#图论模型` — 路径规划、网络流类
- `#微分方程` — 动态系统类
- `#博弈模型` — 竞争决策类
- `#机器学习` — ML方法类

### 算法标签
`#TOPSIS` `#熵权法` `#层次分析法` `#灰色关联` `#灰色预测` `#ARIMA` `#回归分析` `#神经网络` `#遗传算法` `#模拟退火` `#粒子群` `#线性规划` `#K-means` `#DBSCAN` `#主成分分析` `#因子分析` `#蒙特卡洛` `#马尔可夫` `#灵敏度分析`

## 检索逻辑

`/math-model` 在 Step 4 会：
1. 根据问题类型扫描匹配的标签
2. 读取匹配论文的解题思路和模型信息
3. 推荐可用的模型和改进方向
