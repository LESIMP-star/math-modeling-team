# "中国电机工程学会杯"全国大学生电工数学建模竞赛 论文规范

## 基本要求
- **语言**：中文
- **文件格式**：PDF（参赛论文必须是单独的 PDF 文件，不要压缩）
- **支撑材料**：WinRAR 压缩（RAR 或 ZIP，大小不超过 20MB），包含所有可运行源程序、自主查阅的数据资料、较大篇幅中间结果图表等
- **页数限制**：正文控制在 25 页以内（不含附录），附录页数不限
- **字体**：正文宋体小四号字
- **页面**：A4 纸，上下左右各 2.5cm 页边距

## 论文结构

### 第一页：封面页（不编页码）
- 参赛编号（填写错误后果自负）
- 论文题目：A/B 题 + 具体题目（题目不用自拟）

### 第二页：摘要页（从本页开始编页码）
- 论文题目
- 摘要（简明扼要，篇幅不超过一页，无需译成英文）
- 关键词（无需译成英文）
- 页码：页脚居中，阿拉伯数字从 "1" 开始连续编号

### 第三页起：正文（不要目录，控制在 25 页以内）
- 问题分析与模型建立
- 模型求解与结果
- 结果分析与灵敏度分析
- 模型优缺点与推广

### 正文之后：附录（页数不限）
- 可运行源程序代码
- 中间结果图表等支撑材料

## 参考文献格式

正文中引用处用方括号标示编号，如 [1][3] 等；引用书籍还需指出页码。

### 书籍
```
[编号] 作者，书名，出版地：出版社，出版年，页码.
```

### 期刊论文
```
[编号] 作者，论文名，杂志名，卷期号：起止页码，出版年.
```

### 网上资源
```
[编号] 作者，资源标题，网址，访问时间（年月日）.
```

## 禁止事项
- 论文中**不得出现竞赛队员和参赛学校等信息**，否则该论文作废
- 不符合本规范的论文将被视为违反竞赛规则，可能被取消评奖资格
